Data Flow
═════════
User Input → SearchInput
          ↓
     useYouTubeUrl (validation)
          ↓
     isValid → Enable SearchButton
          ↓
     User clicks OR presses Enter
          ↓
     LandingPage.handleSubmit()
          ↓
     [Future: API call to backend]
          ↓
     [Future: Process results]
